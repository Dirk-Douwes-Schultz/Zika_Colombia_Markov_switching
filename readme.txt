Code for paper "Zika epidemic in Colombia and environmental and sociodemographic contributors: an application of a space-time Markov switching model" 
see paper for more details

This repository is still being updated and will eventually contain code to replicate all figures in the paper. Currently it contains code to run the model and check convergence.

zsms_model - Code for the zero-state space-time Markov switching model from the paper. This code will run the model on the Zika data and check convergence 

cases_and_covariables_allColombia_2014-2016 - data
